## üìñ About The Project

**VINTAGE CO.** is a modern, visually stunning eCommerce front-end application tailored for selling authentic, curated vintage clothing. It features a premium dark-mode aesthetic, utilizing sleek glassmorphism panels, stylish micro-animations, and dynamic text gradients to create an immersive, high-end shopping experience.

This project was built from the ground up focusing on performance, modular design, and a pixel-perfect user interface.

- **Premium Aesthetics**: Dark-mode dominant design with smooth glass-like panels over an immersive backdrop.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You will need **Node.js** (v18 or higher recommended) and `npm` installed.

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/nivas4506/VINTAGE.CO-GEMINI-CLI..git
   cd vintage-clothing-ecommerce
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:5173/` to view the application.

---

## üì¶ Building for Production

To create a highly optimized production branch:

```sh
npm run build
```

This will output all static assets into the `/dist` directory, which is ready to be hosted on platforms like Vercel, Netlify, or GitHub Pages.

To preview the created build locally:
```sh
npm run preview
```
